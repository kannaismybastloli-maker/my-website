ຄວາມຮູ້ພື້ນຖານກຽວກັບ Software

ພື້ນຖານທີ່ຈະເຮັດໃຫ້ຮູ້ຈັກ ແລະ ເຂົ້າໃຈກຽວກັບ software ຫຼາຍຂື້ນ

OS (Operating System)

OS ຫຼື ຊື່ເຕັມແມ່ນ Operating System ແມ່ນ ລະບົບປະຕິບັດການ ທີ່ເຮັດໃຫ້ຄອມພີວເຕີທຳງານໃນເບື້ອງຕົ້ນ ເຊັ້ນ: ເຮັດໜ້າທີ່ຄວມຄຸມຄອມພິວເຕີ ແລະ ເຮັດໃຫ້ຜູ້ໃຊ້ເຂົ້າເຖີງການເຮັດວຽກຕ່າງໆໄດ້ງ່າຍ ແລະ ສະດວກຫຼາຍຂື້ນ

ໜ້າທີ່ຫຼັກໆຂອງ OS

  • ຈັດການ Hardware ຕ່າງໆ ເຊັ້ນ: CPU , RAM , Hard disk / SSD , Meyboard , Muose

  • ຮອງຮັບ ແລະ run ໂປຣແກຣມຕ່າງໆ

  • ໃຫ້ Interface ຕ່າງໆ ເຊັ້ນ: Icon , Desktop Scene , Menu ຕ່າງໆ

  • ຈັດການກັບໄຟລ ເຊັ້ນ: ສ້າງ , ລົບ , ຍ້າຍ , ບັນທືກ

OS ຕະກຸນຕ່າງໆ

ໂດຍທົ່ວໄປແລ້ວ OS ຈະແບ່ງເປັນ 3 ຕະກຸນໃຫຍ່:

  • Window : ເປັນ OS ທີ່ນິຍົມຫຼາຍທີ່ສຸດໃນລະບົບຄອມພີວເຕີ ແລ້ວໃຊ້ກັນໂດຍທົ່ວໄປ.
    ຈຸດເດັ້ນຢູ່ທີ່ ໃຊ້ງາຍ , ຮອງຮັບໂປຣແກຣມເກືອບທຸກຢ່າງທີ່ນີ້ຍົມໃຊ້ , ໃຊ້ໄດ້ກັບຄອມຫຼາຍໆລຸ້ນ.
    ພັດທະນາໂດຍ Microsoft ຜູ້ກໍ່ຕັ້ງ BillGate
    window ເຄີຍມີມາຫຼາຍໆ version ມີ່ຜ່ານມາແລ້ວຕົວຢ່າງເຊັ້ນ:

    • window 7

    • window 8

    • window 10

    • window 11

  • Linux : ເປັນ OS ທີ່ເປັນ Open Source ສາມາດດາວໂຫຼດໃຊ້ໄດ້ເລຍ ແລະ ເຮົາຍັງສາມາດຂຽນໂຄ້ດປັບແຕ່ງສ່ວນຕ່າງໆໄດ້ຕາມໃຈທີ່ເຮົາຕ້ອງການໄດ້ເລຍ
    ຈຸດເດັ້ນຂອງ linux ຢູ່ທີ່ ຟີຣ , ເປັນ Open Source ສາມາດແກ້ໄຂໄດ້ , ມີຄວາມປອບໄພສູງ ແລະ ເໝາະສຳຫຼັບ Server and developer
    Linux ຍັງມີຫຼາຍ Distributions ຫຼຶ ຫຼາຍ ໂມເດັວໃຫ້ເລືອກໃຊ້ ຕົວຢ່າງໂຕທີ່ນີ້ຍົມຫຼາຍເຊັ້ນ:

    • Ubuntu

    • Fedora

    • Debian

    • Kali Linux (ໃຊ້ດ້ານ security)

  • Unix ເປັນ OS ທີ່ເກົ່າແຕ່ມີຄວາມໝັ້ນຄົງສູງ ແລະເປັນ ຕົ້ນແບບ ຂອງ OS ຫຼາຍໆຕົວໃນປັດຈຸບັນ ແລະ ຖຶກພັດທະນາມາຕັ້ງແຕ່ດົນ ໂດຍໃຊ້ໃນລະບົບຄອມພິວເຕີຂະໜາດໃຫຍ່ ແລະ server
    ຈຸດເດັ້ນຢູ່ທີ່ ປອດໄພ ແລະໝັ້ນຄົງສູງ , ຮອງຮັບຜູ້ໃຊ້ຫຼາຍຄົນພ້ອມກັນ (Multi-user) , ເໝາະກັບ server ແລະລະບົບໃຫຍ່ , ເປັນພື້ນຖານຂອງ OS ຫຼາຍຕົວ

  • ນອກຈາກນີ້ຍັງມີ OS ທີ່ໄດ້ພັດທະນາ ແລະ ຕໍ່ຍອດ ມາຈາກ 3 ຕະກຸນນີ້ອີກດ້ວຍເຊັ້ນ:

    • macOS / IOS ໄດ້ມີການນຳ Unix/BSD ມາເປັນຖານໃນການພັດທະຍາແລ້ວຕໍ່ຍອດ

    • Android ໃຊ້ Linux Kernel ເປັນຫົວໃຈຂອງລະບົບໃນການພັດທະນາຕໍ່ມາ

ພາກສ່ວນແນະນຳ

ຈະເປັນຄຳແນະນຳ ແລະ ບົບຮຽນເພີ່ມເຕີມ
