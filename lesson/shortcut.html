Short Cut ຫຼື ຄີລັດ

ຄີລັດຕ່າງໆທີ່ຈະມາຊ່ວຍອຳນວຍຄວາມສະດວກໃນການຮຽນ ແລະ ການເຮັດວຽກຕ່າງໆ

ຄີລັດຕ່າງໆທີ່ຈະມາແນະນຳ

ຄີລັດເລົ່າຈະຊ່ວຍໄດ້ຫຼາຍໃນການເຮັດວຽກໄດ້ ສະດວກສະບາຍ ຫຼາຍຂື້ນ

*ສາມາດເບີງປຸ້ມທີ່ຈະກົດໄດ້ໃນຂ້າງຊ້າຍຈໍ*
ປຸ່ມທີ່ກົດ ສີ່ງທີ່ໄດ້ ວິດີໂອຕົວຢ່າງ
Ctrl + Cເປັນການຄັດລອກສີ່ງທີ່ເລື່ອກ ຫຼື ທາສີໄວ້
Ctrl + Vເປັນການວ່າງຂໍ້ມູນທີ່ເຮົາຄັດລອກມາ
Ctrl + Aເລືອກຂໍ້ຄວາມທັ້ງໝົດທີ່ມີໃນໜ້າຈໍ
Ctrl + Xເປັນການຕັດຂໍ້ຄວາມທີ່ເຮົາເລືອກໄວ້ ແລະ ສາມາດເອົາໄປວາງໄດ້
Ctrl + Zເປັນການຍ້ອນກັບສີ່ງທີ່ເຮົາເຮັດກ່ອນ້ານີ້ ສາມາດໃສ່ໄດ້ກັບ ຫຼາຍຢ່າງເຊັ້ນ: ການພີມ ຫຼື ການລົບໄຟຕ່າງໆ
Ctrl + Yເປັນການຍ້ອນກັບສີ່ງທີ່ Ctrl + Z ເຮັດ ສ່ວນຫຼາຍໃຊ້ໃນການພິມເອກະສານ
Window + Dເປັນການພົບໜ້າ ຫຼື ຫຍໍ້ໜ້າຕ່າງຂອງໂປຣແກຮມທີ່ເຮົາໃຊ້ຢູ່ໄປຫາໜ້າ Desktop ແລະ ກົດອີກຄັ້ງເພື່ອກັບມາໜ້າໂປຣແກຣມຄືເກົ່າ
Alt + Tabເປັນການສະລັບ ລະວ່າງ ໂປຮແກຣມແຕ່ລະໂປຣແກຣມ ຫຼຶ ເລື່ອກໂປຣແກຣມທີ່ເຮົາເປິດໄວ້ ຖ້າກົດບາດ tab ບາດດຽວ ຈະເປັນການສະຫຼັບແບບໄວໆ ແຕ່ຖ້າ ກົດ tab ຫຼາຍເທື່ອ ຈະເປັນການເລືອກໂປຣແກຣມ
Ctrl + Shift + Nເປັນການສ້າງໄຟລໄໝ່ໃຊ້ໄດ້ທັ້ງໃນ ໜ້າ desktop ແລະ ໃນ ໂປຣແກຣມ File Explorer
window key + Eເປິດໂປຣແກຣມ File Explorer
window key + Iເປິດ Setting
Ctrl + Shift + ESCເປິດ Task Manager
Window key + PrtScບັນທຶກໜ້າຈໍ້ທັ້ງໜ້າຈໍ
Window key + Shift + Sບັນທຶກໜ້າຈໍ້ແຕ່ສາມາດເລືອກໄດ້ວ່າຈະບັນທຶກສ່ວນໃດ ແລະ ມັນຈະຢູ່ໃນ Clipborad ແລະ ເຮົາສາມາດວ່າງໄດ້ເລຍ
Alt + F4ຖ້າຢູ່ໜ້າ desktop ຈະເປັນການ Shut Down ຖ້າຢູ່ໃນໂປຣແກຣມໃດໜື່ງຈະເປັນການປິດໂປຣແກຣມນັ້ນ
Window key + Vຈະເປັນການເປິດ tap Clipborad ຂື້ນມາເພື່ອເບີ່ງສິ່ງທີ່ເຮົາເຄີຍຄັດລອກໄວ້

(ໝາຍເຫດ: ຖ້າຍັງບໍ່ເຄຍເປິດຕ້ອງໄດ້ກົດເປິດກ່ອນແລ້ວຈະໃຊ້ໄດ້)
Window key + Spacebarເປັນການປຽນພາສາທີ່ໃຊ້ໃນການພີມ
Window key + Alt + Dເປັນການສະແດງໜ້າສະແດງວັນ , ເດືອນ , ປີ ແລະ ເວລາ
Window key + Ctrl + Dເປັນການເພີ່ມ ໜ້າຈໍ Desktop ໃໝ່ຂື້ນມາ
Window key + . (ໃ)ຮ້ອງໃຊ້ແຖບ emoji
Window key + Xການເປິດເມນູ ການຕັ້ງອຸປະກອນຕ່າງໆຂື້ນມາໃຫ້ໃຊ້ ເຊັ້ນ: setting , Task manager , File Explorer ແລະ ອື່ນໆ ໃຫ້ສາມາດເລື່ອກໃຊ້ໄດ້ຕາມຕ້ອງການ
F11ແມ່ນການເຮັດໃຫ້ໂປຣແກຣມນັ້ນເຕັມຈໍ ແລະ ກົດອິກເທື່ອໜື່ງເພື່ອກັບຄືນ
Shift + delແມ່ນການລົບແບບ ຖາວອນ ການລົບທົ່ວໄປຈະລົບແລ້ວຈະລົບແລ້ວໄປຢູ່ໃນ Recycle Bin
Window key + Rແມ່ນການ ຮ້ອງເມນູປ້ອນຄຳສັ່ງພີເສດຂື້ນມາເພື່ອປ້ອນຄຳສັ່ງຕ່າງໆໄດ້
hold Shitf + ຕົວກະພີບການເລືອກທາຂໍ້ຄວາມທີ່ຫຼາຍ ແລະ ຍາວແບບງ່າຍຂື້ນ

ພາກສ່ວນແນະນຳ

ຈະເປັນຄຳແນະນຳ ແລະ ບົບຮຽນເພີ່ມເຕີມ

ການຝຶກໃຊ້ໃຫ້ເປັນປະຈຳ

ການທີ່ໃຊ້ ຄິລັດພວກນີ້ໄດ້ຈະຊ່ວຍໃຫ້ການຮຽນ , ການຂຽນໂປຣແກຣມ ຫຼື ການເຮັດສິ່ງຕ່າງໆດ້ວຍຄອມພີວເຕີ ນັ້ນວ່ອງໄວຂື້ນຫຼາຍ ສະນັ້ນໃຊ້ເຖາະຖືວ່າຂໍ້
