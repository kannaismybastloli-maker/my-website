ບົບຮຽນທົ່ວໄປກຽວກັບ Microsoft Word

ຈະເປັນການບອກກຽວກັບ ພື້ນຖານຕ່າງໆຂອງ word ເຊັ້ນເມນູຕ່າງໆ ແລະ ການໃຊ້ໂດຍພື້ນຖານ ເພື້ອໃຫ້ທຸກຄົນສາມາດນຳໃຫ້ ແລະ ຕໍ່ຍ່ອດໃນສ່ວນຕ່າງໆໄດ້

ເມນູຕ່າງໆໃນ Microsoft Word

ຈະມີຢູ່ 3 ພາກສ່ວນຫຼັກໆຄື:

  • ພາກສ່ວນ title bar:
    ຈະເປັນພາກສ່ວນທີ່ມີການຢູ່ກາງໃນ ແຖວ ເມນູດ້ານເທິງທີ່ຈະບອກຊື່ໄຟລ ນາມສະກຸນ ແລະ ພື້ນທີ່ເກັບວ່າເກັບຢູ່ໃສ່

  • ພາກສ່ວນ Tool bar:
    ຈະເປັນພາກສ່ວນທີ່ຢູ່ແຖວດຽວກັບ title bar ແຕ່ຈະຢູ່ທາງດ້ານຂວາ ຈະມີອຸປະກອນຕ່າງໆໃຫ້ໃຊ້ເຊັ່ນ: save , undo , redo , new file ເປັນຕົ້ນ ແລະ ເຮົາສາມາດປັບແຕ່ງເພີ່ມເຕີມໄດ້

  • ພາກສ່ວນ Menu bar ຫຼື : Ribbon bar
    ຈະເປັນພາກສ່ວນທີ່ຢູ່ໃນຈະມີ ເມນູເຄື່ອງເມື່ອຕ່າງໆຢູ່ຫຼາຍໆຢ່າງເຊັ້ນ: Home , Insert , Design , layout ແລະ ອື່ນໆທີ່ສາມາດເອົາອອກ ຫຼື ເພີ່ມເຂົ້າໄດ້ອິກ

ຄວາມຮູ້ພື້ນຖານກຽວກັບສີ່ງທີ່ເຄື່ອງມື່ຕ່າງໆ ໃນ menu bar ສາມາດເຮັດໄດ້ ຫຼື ມີໜ້າທີຫຍັງ

File

ໜ້າທີ່ຫຼັກໆຂອງ File ຈະມີດັ້່ງນີ້:

  • New ເພີ່ມເອກະສານໃໝ່

  • Open ເປິດເອກະສານເກົ່າ

  • Save, Save as ບັນທຶກເອກະສານ

  • Print ພີມເອກະສານອອກມາທາງເຄື່ອງປິ້ນ

Home

ໜ້າທີ່ຫຼັກໆຂອງ Home ຈະມີດັ້່ງນີ້:

  • ປ່ຽນຟອນ, ປ່ຽນຂະໜາດຟອນ, ປຽນສີຟອນ, ປັບຕໍາແໜ່ງ (ຊ້າຍ,ກາງ, ຂວາ, ສອງຂ້າງ)

  • ທໍາລາຍການ (Bullets, Numbering)

  • ເລືອກ Style haeding ຕ່າງໆ

Insert

ໜ້າທີ່ຫຼັກໆຂອງ Insert ຈະມີດັ້່ງນີ້:

  • ແຊກຮູບ (picture)

  • ແຊກຕາຕະລາງ (Table)

  • ເພີ່ມ Header, footer

  • ແຊກຮູບຮ່າງ Shapes

Design

ໜ້າທີ່ຫຼັກໆຂອງ Design ຈະມີດັ້່ງນີ້:

  • ປ່ຽນຮູບແບບເອກະສານ Themes

  • ປ່ຽນພຶ້ນຫຼັງ, ຂອບໜ້າ

  • ການໃຊ້ ລາຍນໍ້າ ຫຼື water mark ຕ່າງໆ

Layout

ໜ້າທີ່ຫຼັກໆຂອງ Layout ຈະມີດັ້່ງນີ້:

  • ປ່ຽນຂອບໜ້າ (Margins)

  • ປ່ຽນເປັນລວງຕັ້ງ, ລວງນອນ (Portrait / Landscape)

  • ແບ່ງເປັນຖັນ Column

References

ໜ້າທີ່ຫຼັກໆຂອງ References ຈະມີດັ້່ງນີ້:

  • ສ້າງສາລະບັນ (Table of content)

  • ເພີ່ມ footnote

  • ການອ້າງອິງ (Citations)

Mailings

ໜ້າທີ່ຫຼັກໆຂອງ Mailings ຈະມີດັ້່ງນີ້:

  • ສ້າງຈົດໝາຍອັດຕະໂນມັດ (Mail merge)

  • ຊ່ອງຈົດໝາຍ (Envelops)

  • ປ້າຍຊື່ (Labels)

Review

ໜ້າທີ່ຫຼັກໆຂອງ Review ຈະມີດັ້່ງນີ້:

  • ກວດຄຶນຄຳຜິດ (Spelling & grammar)

  • ເພີ່ມ Comment

  • Track changes ກວດຄຶນການແກ້ໄຂ

view

ໜ້າທີ່ຫຼັກໆຂອງ view ຈະມີດັ້່ງນີ້:

  • ປຽນຮູບແບບການເບິງເອກະສານ (Print layout, Web layout)

  • ປິດ, ເປິດ ruler, Gridlines, Navigation Pane

ສະຫຼຸບລວມສີງທີ່ Microsoft word ສາມາດເຮັດໄດ້

ສີ່ງທີ່ microsoft word ສາມາດເຮັດໄດ້ນັ້ນມີຫຼາຍໆຢ່າງ ແຕ່ຖ້າໃຫ້ເຮົາສະຫຼຸບມາຈະໄດ້ດັ້ງນີ້:

  • ສາມາດລົບ, ຍົກຍ້າຍ, ສໍາເນົາ ຂໍ້ຄວາມໄດ້

  • ສາມາດຂຽນຕົວໜັງສືໜາ, ເນີ້ງໄດ້

  • ສາມາດປັບຂະໜາດຕົວໜັງສຶໄດ້

  • ສາມາດແກ້ໄຂເອກະສານໄດ້

  • ສາມາດຂຽນຕົວໜັງສືປະເພດຕ່າງໆໄດ້

  • ສາມາດໃສ່ຮູບໃນເອກກະສານໄດ້

  • ສາມາດສ້າງຕາຕະລາງໄດ້

  • ສາມາດກວດຄຶນຄຳຜິດ ແລະ ແກ້ໄຂໄດ້

ພາກສ່ວນແນະນຳ

ຈະເປັນຄຳແນະນຳ ແລະ ບົບຮຽນເພີ່ມເຕີມ

ຄວນຮຽນໃຊ້ໃຫ້ຄ່ອງແຄ້ວ ແລະ ຖະໜັດ

ເພາະ microsoft word ນັ້ນເປັນທີ່ນຳໃຊ້ຫຼາຍສຳຫຼັບໂປຣແກຣມ ເຮັດເອກະສານ ຫຼື ຂຽນເອກະສານ ແລ້ວຖ້າເກັ່ງໃນການຂຽນເອກະສານຕ່າງໆຈະຊ່ວຍໃນການຮຽນ ແລະ ການເຮັດວຽກຂອງເຮົາໄດ້ຫຼາຍ
