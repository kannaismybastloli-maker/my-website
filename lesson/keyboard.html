ຄວາມຮູ້ພື້ນຖານກຽວກັບ Keyboard

ພື້ນຖານທີ່ຈະເຮັດໃຫ້ຮູ້ຄວາມສາມາດ ແລະ ໜ້າທີ່ຂອງປຸ່ມຕ່າງໆໃນ keyboard

ປຸ່ມຕ່າງໆ ໂດຍທົ່ວໄປ

ປຸ່ນຕ່າງໆ ແລະ ຄວາມສະມາດ ແລະ ຕົວຢ່າງການໃຊ້

ປຸ່ມ ຕົວປຸ່ມ ການນຳໃຊ້
ຕົວອັກສອນA-Z / ກ-ຮ ພີມຕົວໜັງສືທົ່ວໄປ
ປຸ່ມຕົວເລກ0-9ພີມຕົວເລກທົ່ວໄປ
ປຸ່ມ FunctionF1-F12ຄຳສັ່ງພີເສດໃນແຕ່ລະໂປຣແກຣມເຊັ້ນ: F5 Refresh ໃນໜ້າ desktop
ປຸ່ມ ControlCtrl , Alt , Shitfໃຊ້ກົດກັບປຸ່ມອື່ນເຊັ້ນ:

  • Ctrl + C : Copy

  • Alt + F4 : Shut Down

  • Shift + ຕົວອັກສອນ : ຈະໄດ້ຕົວພີມໃຫຍ່
ປຸ່ມ EnterEnterກົດເພື່ອລົງແຖວໃຫຍ່
ປຸ່ມ SpacebarSpaceຍະວ່າງ
ປຸ່ມ BackspaceBackspaceລົບຂໍ້ຄວາມດ້ານຫຼັງ ຫຼຶ ລົບແຕ່ຂວາຫາຊ້າຍ
ປຸ່ມ DeleteDelete / Delລົບຂໍ້ຄວາມດ້ານໜ້າ ຫຼຶ ລົບແຕ່ຊ້າຍຫາຂວາ
ປຸ່ມ Arrow↑ ↓ ← →ເລື່ອນ Cursor ຫຼື ຕົວກະພີບ
ປຸ່ມ EscEscອອກຈາກໂປຣແກຣມ ຫຼື ຍົກເລີກຄຳສັ່ງ
ປຸ່ມ TabTabເລື່ອນໄປຊ່ອງຖັດໄປຄ້າຍໆກັບ space ແຕ່ຫຼາຍກວ່າ
ປຸ່ມ Caps LockCaps Lockປຽນຈາການພີມຕົວໜ້ອຍເປັນພິມຕົວໃຫຍ່ ແລະ ພີມຕົວໃຫຍ່ເປັນຕົວໜ້ອຍ
ປຸ່ມ Windows⊞ Winເປີດ Start Menu
ປຸ່ມ NumLockNumLockລ໊ອກ Numpad ໄວ້
ປຸ່ມ Symbol@ , # , $ , ແລະ ອຶ່ນໆສັກຍາລັກຕ່າງໆ
ປຸ່ມ PrtSc SysRqPrtSc SysRqກົດ win + PrtSc SysRq : ຈະເປັນການບັນທຶກໜ້າຈໍ
ປຸ່ມ HomeHomeເປັນການເລື່ອນ Corsor / ຕົວກະພີມ ຫຼື ວິດີໂອ ໄປທີ່ຈຸດເລີມຕົ້ນ
ປຸ່ມ EndEndເປັນການເລື່ອນ Corsor / ຕົວກະພີມ ຫຼື ວິດີໂອ ໄປທີ່ຈຸດສິນສຸດ
ປຸ່ມ Pg UpPg Upເລື່ອນໜ້າຈໍຂື້ນ
