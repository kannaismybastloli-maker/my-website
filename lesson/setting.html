ການຕັ້ງຄ່າ (Setting) ທົ່ວໄປ

ຈະຊ່ວຍໃຫ້ຮູ້ຈັກຄວາມພີວເຕີຫຼາຍຂື້ນ ແລະ ສາມາດປັບແຕ່ງສີ່ງຕ່າງໆໃຫ້ເຂົ້າກັບການໃສ່ງານຄອມພີວເຕີຂອງເຮົາໄດ້

ຕັ້ງຄ່າຕ່າງໆກຽວກັບ ພາສາ

ການຕັ້ງຄ່າພາສາ ທີ່ຈະມາແນະນຳແມ່ນຈະມີການ ເພີ່ມ ແລະ ການລົບພາສາ

  • ຂັ້ນຕອນການເພີ່ມພາສາຕ່າງໆ:

    ໄປທີ່ຕັ້ງຄ່າ setting → ເລືອກ Time and Language → ເລືອກ languege ໃນ side menu → ກົດ Add a language → ຄົ້ນຫາ ແລະ ເລືອກພາສາທີ່ຕ້ອງການ → ກົດ Next → ກົດ Install

  • ຂັ້ນຕອນການລົບພາສາຕ່າງໆ:

    ໄປທີ່ຕັ້ງຄ່າ setting → ເລືອກ Time and Language → ເລືອກ languege ໃນ side menu → ແລ້ວຈະເຫັນ ພາສາທັ້ງໝົດທີ່ເຮົາມີ → ຖ້າຈະລົບພາສາໃດໃຫ້ຄິກໃສ່ → ແລ້ວກົດ Remone ໄດ້ເບຍ

ການຕັ້ງຄ່າ ວັນທີ່ ແລະ ເວລາ

ການຕັ້ງ ວັນທີ່ ແລະ ເວລາໃນນີ້ຈະມາແນະນຳການຕັ້ງຄ່າວັນທີ່ ແລະ ເວລາ ອັດຕະໂນມັດ ແລະ ຕັ້ງຕາມໃຈ ແລະ ການຈັດຕໍາແໜ່ງ ຂອງວັນທີ່, ເດືອນ, ປີ

  • ການຕັ້ງຄ່າວັນທີ່ແບບອັດຕະໂນມັດ

    ໄປທີ່ຕັ້ງຄ່າ setting → ເລືອກ Time and language → ເລືອກ Date and time ໃນ side menu → ຕິກເປິດ set time automatically ຖ້າຕ້ອງການໃຫ້ມັນປັບເວລາໃຫ້ ອັດຕະໂນມັດ ຫຼື ຖ້າຈະຕັ້ງເວລາເອງໃຫ້ຕິກປິດ ແລະ ຕັ້ງເວລາາໃນ Set date and time manually ກົດ change ແລ້ວຕັ້ງຕາມໃຈໄດ້ເລຍ

  • ການຕັ້ງຄ່າ ຈັດຕຳແໜ່ງຂອງວັນທີ່ເດືອນປີ

    ໄປທີ່ Control Panel → ເລືອກ Clock and Region → ກົດ Date and time → ກົດ Change date and time → Change calendar setting → ແລ້ວກົດ Additional settings → ເລຶອກ time ໃນ ເມນູທາງເທີງ → ປຽນໃນແຖບເມນູ Date Formats → ກົດ Apply → ກົດ Ok
    ໃນການຂຽນຕົວອັກສອນນັ້ນ d : ວັນ , M : ເດືອນ , y : ປີ

ການຕັ້ງຄ່າ Mouse

ການຕັ້ງຄ່າເມົ້າທີ່ນະມາເນະນຳຈະມີ: ຄວາມໄວເມົ້າ , ຄິກຊາຍ/ຂວາ , ຮູບແບບການຄິກ , ຂະໜາດ ແລະ ສີເມົ້າ

  • ການຕັ້ງເມົ້າ ໂດຍພື້ນຖານ

    ໄປທີ່ຕັ້ງຄ່າ setting → ເລືອກ Devices → ເລືອກ Mouse ໃນເມນູຂ້າງຊາຍ → ແລ້ວຈະມີ:

    • Select your primary button: left / right
      ເປັນການປັບໃຫ້ຄິກໃດໜື່ງເປັນຄິກປົກກະຕິ ຫຼື ຄິກຫຼັກ ຕາມຄ່າທົ່ວໄປແມ່ນ ຊ້າຍ

    • Cursor speed
      ຈະເປັນການປັບຄວາມໄວ Cursor ເມົ້າ ຕາມຄ່າເລີ່ມຕົ້ນແມ່ນ 10

    • Choose how many lines to scroll each time
      ຈະເປັນການປັບຄວາມໄວໃນການໃຊ້ລູກກິ້ນເລື່ອນໜ້າເວັບ ຫຼື ໜ້າຈໍຂອງແຕ່ລະໂປຣແກຣມ

  • ການປັບແຕ່ງເມົ້າ ໂດຍພື້ນຖານ

    ໄປທີ່ຕັ້ງຄ່າ setting → ເລືອກ Devices → ເລືອກ Mouse ໃນເມນູຂ້າງຊາຍ → ແລ້ວເລືອກ Adjust mouse & Cursor size ແລ້ວຈະມີການຕັ້ງຄ່າຄື:

    • Change pointer size: ເປັນການປຽນຂະໜາດເມົ້າ

    • Change pointer size: ການປັບສີຂອງເມົ້າ

  • ການປັບແຕ່ງເມົ້າ click ຫຼື double click

    ໄປ This pc → ເລື່ອກ view ໃນ ເມນູທາງເທິງ → ເລືອກ options → ກົດ Change folder search options → ປຽນໃນແຖບ menu Click item as follows ຈະມີ:

    • Single-click to open an item (point to select): ເປັນການກົດ ຄິກເທືອດຽວເພື່ອເປິດ

    • Double-click to opne an item (sigle-click to select): ເປັນການກົດຄິກ ສອງເທື່ອເພື່ອເປິດ (ຄ່າເລີ່ມຕົ້ນ)

ການຕັ້ງຄ່າ Taskbar

ການຕັ້ງຄ່າ taskbar ທີ່ຈະມາແນະນຳຈະມີ: ການປຽນຕຳແໜ່ງ , ການລົງໄປອັດຕະໂນມັດ ແລະ ການເພີ່ມ icon ຕ່າງໆລົງໃນ taskbar

ຂັ້ນການໄປຕັ້ງຄ່າ ແລະ ປັບແຕ່ກ taskbar

ຄິກຂວາທີ່ taskbar → ເລືອກ Taskbar setting → ແລ້ວຈະເຫັນເມນູຕ່າງໆດັ້ງນີ້:

  • Lock the taskbar : ຖ້າເປິດໄວ້ມັນຈະລ໊ອກ taskbar ໄ້ວ ເຮົາຈະບໍ່ສາມາດຍ້າຍຕຳແໜ່ງໄດ້

    ແຕ່ຖ້າປິດເຮົາຈະສາມາດຍ້າຍຄຳແໜ່ງຂອງ taskbar ໄດ້

  • Automatically hide the taskbar in table mode
    ຈະເປັນການ ໃຫ້ taskbar ສ້ອນລົງໄປເອງ ແລ້ວເວລາເຮົາເອົາເມົ້າໄປຈີ້ໃສ່ມັນຈະຂື້ນມາ

  • ໃນແຖວເມນູ Notification area → ແລ້ວກົດ select which icon appear on the taskbar
    ເປັນການພີ່ມ setting icon ຕ່າງໆເຂົ້າມາໃນ taskbar ຂອງເຮົາ
