ຄວາມຮູ້ພື້ນຖານກຽວກັບ Hardware

ພື້ນຖານທີ່ຈະເຮັດໃຫ້ຮູ້ກຽວກັບ Hardware ຕ່າງໆ

Hardware ປະກອບມີ:

  1. Case: ແມ່ນເຄື່ອງຄອມພິວເຕີ ຫຼື ກ່ອງທີ່ໃຊ້ໄວ້ໃນການໃສ່ສ່ວນປະກອບຕ່າງຂອງຄອມ

    ຈະເປັນທີ່ເອົາໄວປະກອຍສ່ວນຕ່າງໆຂອງຄອມພີວເຕີເຊັ້ນ:

    Main Board , RAM , ROM ແລະ ເປັນບ່ອນຕໍ່ Port ຕ່າງໆ

  2. Speaker: ລຳໂພງ

    ລຳໂພງຈາກຄອມຖ້າເຮົາແບ່ງປະເພດງ່າຍໆຈະມີໄດ້:

    • Built-in Monitor Speakers: ລຳໂພງທີ່ມີມາກັບຈໍ

    • Internal Pc Speaker: ລຳໂພງທີມີມາກັບ Case

    • Exteranl Speakers: ລຳໂພງພາກນອກຕ່າງໆ

  3. Microphone: ໄມໂຄໂຟນທີ່ໃຊ້ໃນການຮັບສຽງ

    ລຳໂພງຈາກຄອມຖ້າເຮົາແບ່ງປະເພດງ່າຍໆຈະມີໄດ້:

    • Built-in Microphone: ໄມໂຄໂຟນທີ່ມີມາກັບຈໍ

    • Internal Microphone: ໄມໂຄໂຟນທີມີມາກັບ Case
      (ປົກກະຕິບໍ່ຄ່ອຍມີແຕ່ຍັງມີບາງເຄື່ອງທີ່ມີ)

    • Exteranl Microphone: ໄມໂຄໂຟນພາກນອກຕ່າງໆ

  4. RAM: Random Access Memory ໜ່ວຍຄວາມຈຳສຳຮອງຊ່ວຍຄາວ

    ໜ້າທີ່ຄືຊ່ວຍໃຫ້ໂປຣແກຣມຕ່າງໆເປິດໄວຂື້ນ ສາມາດເສີມໃຫ້ມັນຫຼາຍຂື້ນໄດ້ເພື່ອໃຫ້ຄອມເຮົາເຮັດວຽກໃຫ້ມີປະສິດຕີພາບໄດ້

  5. CPU: Central Processing Unit ແມ່ນໜ່ວຍປະມວນຜົນຂອງຄອມ

    ໜ້າທີ່ຄືຄວບຄຸມການເຮັດວຽກ ຂອງອຸປະກອນອື່ນໆ ໃນຄອມພິວເຕີ

  6. Keyboard: ແປ້ນພີມ

    Keyboard ຍັງມີຫຼາຍຂະໜາດເຊັ້ນ:

    • Full-size 100% : ມີປຸ່ມຄົບ
      F-row , numpad ເປັນຮູບແບບມານຕະຖານ ເໝາະສຳຫຼັບວຽກທີ່ຕ້ອງການປະສິງທີພາບສູງ

    • Compact Full-Size 96% : ມີປຸ່ມເກືອບຄົບ
      F-row, numpad ແຕ່ກະທັດຮັດກວ່າ 100%
      ໃຊ້ໄດ້ຄືກັບ Full-Size ແຕ່ກະທັດຮັບ , ປະຍັດພື້ນທີ່ຫຼາຍກ່ວາ

    • TKL Keyboard 80% : ຕັດ numpad ດ້ານຂວາອອກ ແຕ່ຍັງມີປຸ່ມ F-row ຢູ່ ກະທັດຮັດເໝາະສຳຫຼັບການພັກພາໄດ້ສະດວກຫຼາຍຂື້ນ ແຕ່ຍັງຄົງມີປະສິດຕິພາບໃນການເຮັດວຽກຢູ່

    • 75% : ຈະເປັນແບບທີ່ຄ້າຍໆກັບ TKL ແຕ່ກະທັດຮັດຫຼາຍຂື້ນ ຍັງມີແຖວ F-row ຢູ່ ກະທັດຮັດຫຼາຍຂື້ນ ເປັນຮູບແບບ keyborad ທີ່ laptop ສະໄຫມໄໝ່ໃຊ້ທົ່ວໄປ

    • 60% : ຈະເປັນແບບທີ່ສະທັດຮັບ ແລະ ພັກພາສະດວກຫຼາຍ ເໝາະສຳຫຼັບການຫຼີນເກມ ຫຼື ເຮັດວຽກທີ່ບໍ່ລະອຽດຫຼາຍ ບໍ່ເໝາະສຳຫຼັບການເຮັດວຽກທີ່ຕ້ອງການຄວາມປະສິດຕິພາບສູງ

  7. Mouse: ເຮັດໜ້າທີ່ໃນການເຄື່ອນໄຫວ Cursor ໃນຈໍ ເພື່ອອອກຄຳສັ່ງຕ່າງໆເຊັ້ນ: ເລື່ອກ , ເປິດ ຫຼື ສັ່ງໂປຮແກຣມຕ່າງໆ

    Mouse ໂດຍທົ່ວໄປມີ 3 ປຸ່ມ :

    • Left click : ຄິກຊາຍ
      (click ທົ່ວໄປ ຈະແມ່ນການ ແລະ bouble click ຈະແມ່ນການຄິກຊ້າຍສອງເທື່ອຕິດໆກັນ)

    • Right click : ຄິກຂວາ
      (ສ່ວນໃຫຍ່ຈະເປັນການ ຮ້ອງເມນູຕ່າງໆຂື້ນມາ)

    • Scroll Wheel : / MIddle Click ລໍ້ເລື່ອນ ຫຼື ຄິກປຸ່ມກາງ
      (ປຸ່ມການສ່ວນໃຫຍ່ຈະເປັນການເປິດໜ້າໃນບາງ Program)

  8. CD-ROM Drive: Compact Disc recobbled

    ເຮັດໜ້າທີ່ໃນການອ່ານຂໍ້ມູນຈາກ CD ແລະ ເອົາມາສະແດງຜົນ
    (ແຕ່ໃນສະໄໝນີ້ສ່ວນຫຼາຍບໍ່ໄດ້ໃຊ້ກັນແລ້ວ)

  9. Printer: ເຄື່ອງປີ້ເອກະສານ

    ແມ່ນເຄື່ອງມືພີມຂໍ້ມູນຕ່າງໆເຊັ້ນ: ຮູບ, ເອກະສານ, ວຽກຕ່າງໆ ອອກມາຈາກຄອມພີວເຕີ

  10. Ports : ແມ່ນຊ່ອງສຽບ ຫຼື ຈຸດເຊື່ອມຕໍ່ຢູ່ໃນຄອມພິວເຕີ ຫຼື ຢູ່ໃນ case ໃຊ້ສຳຫຼັບເຊື່ອມຕໍ່ອຸປະກອນອື່ນໆເຂົ້າກັບຄອມພີວເຕີ

    ໃຊ້ເຊື່ອມຕໍ່ລະຫວ່າງຄອມພິວເຕີ ເຂົ້າກັບອຸປະກອນຕ່າງໆເຊັ້ນ ຈໍ , Mouse , keyboard , ຫູຟັງ , ລຳໂພງ , USB ຕ່າງໆ ແລະ ອື່ນໆ

    ຕົວຢ່າງຂອງ ports ຕ່າງໆ :

    • USB Ports : ສຽບແຟລໄດຣຟ , Mouse , Keyboard

    • HDMI Ports : ໃຊ້ໃນການຕໍ່ຈໍຄອມ ຫຼື ຕໍ່ຈໍ ໂທລະທັດ

    • Audio Ports : ຕໍ່ຫູຟັງ ຫຼື ລຳໂພງ

    • LAN Ports : ຕໍ່ internat

    • Charging Ports : ສຽມສາຍສາກ ຫຼື ສາຍໄຟຕ່າງໆ

  11. Monitor: ໜ້າຈໍສະແດງ

    ໜ້າທີ່ຄືສະແດງຜົນຂໍ້ມູນຕ່ງໆອອກມາໃຫ້ເຫັນໄດ້

    ປະເພດຂອງຈໍ ໂດຍທົ່ວໄປ :

    • LCD : ຈໍບາງ , ປະຢັດໄຟ

    • LED : ພາບຄົມຊັດ , ນິຍົມໃຊ້ຫຼາຍ

    • OLED : ສີງາມ , ຄຸນນະພາບສູງ

  12. Main board ຫຼື Marther board : ແຜນວົງຈອນຫຼັກຂອງຄອມພິວເຕີ

    ເປັນ board ທີ່ເຊື່ອມຕໍ່ອຸປະກອນເອເລັກໂຕນິກທຸກຢ່າງໃນຄອມເຂົ້າກັນ ເຊັ້ນ:

    • CPU

    • RAM

    • Hard Disk / SSD

    • Hard Disk / SSD

    • Graphics Card

    • Keyboard

  13. Expansion board ຫຼຶ Ecpansion Card

    board ເສິມຕ່າງໆທີ່ຈະຊ່ວຍໃຫ້ຄອມພິວເຕີເຮັດວຽກໄດ້ດີຂື້ນ

    ຕົວຢ່າງຂອງ Expansion board :

    • Graphics Card / GPU : ກາດຈໍ່
      ຊ່ວຍໃຫ້ພາບງາມຂື້ນ ແລະ ຫຼີນເກມໄດ້ດີຂື້ນ

    • Sound Card : ກາດສຽງ
      ຊ່ວຍໃຫ້ສຽງດີຂື້ນ

    • Network Card / LAN Card : ກາດເນັດ
      ເຮັດໃຫ້ການເຊື່ອມຕໍ່ອິນເຕີເນັດໄດ້ດີຂື້ນ

  14. Power Supply / PSU : ອຸປະກອນຈາຍໄຟໃຫ້ຄອມພີວເຕີ

    ແມ່ນອຸປະກອນທີ່ແປງໄຟເຮືອນໃຫ້ເຂົ້າກັບຄອມພິວເຕີ ແລະ ສົ່ງໄຟໄປໃຫ້ CPU , Main Board , Hard Disk / SSD , Monitor ແລະ ອື່ນໆ

  15. Hard Disk / SSD : ອຸປະກອນເກັບຂໍ້ມູນຂອງ ຄອມພີວເຕີຕ່າງໆຂອງຄອມພີວເຕີ

    ແມ່ນອຸປະກອນທີ່ແປງໄຟເຮືອນໃຫ້ເຂົ້າກັບຄອມພິວເຕີ ແລະ ສົ່ງໄຟໄປໃຫ້ CPU , Main Board , Hard Disk / SSD , Monitor ແລະ ອື່ນໆ

    ຄວາມແຕ່ກຕ່າງລະວ່າງ HDD ຫຼຶ Hard Disk ກັບ SSD

    • HHD : (Hard Disk Drive) ຈະເປັນແບບຈານໝຸນ ແລະ ຫົວອ່ານ ມີລາຄາທີ່ຖຶກກວ່າ ແລະ ຈະໄດ້ພື້ນທີ່ຈັດກັບຫຼາຍ
      ເໝາະສຳຫຼັບການເກັບ ວິດີໂອ , ຮູບພາບ ແລະ ໄຟລ back up ຕ່າງໆ

    • SSD : (Solid State Drive) ເປັນແບບໃຊ້ chip ໃນການເກັບຂໍ້ມູນ ຄ້າຍໆກັບ USB ແຕ່ໄວກວ່າ
      ເໝາະສຳຫຼັບ ລົງ window ເພື່ອໃຫ້ຄອມເປິດໄວຂື້ນ ແລະ ລົງໂປຣແກຣມເຮັດວຽກຕ່າງເພື່ອໃຫ້ການເຮັດວຽກລື່ນໄຫຼຂື້ນ ແລະ SSD ຍັງມີ 2 ແບບຄື:

      • ແບບ M.2 ຕ້ອງປຽນ ກັບ HDD ຖ້າມີ HDD ຢູ່
      • ແບບ sata ສາມາດ ສຽບໃສ່ບ່ອນໃສ່ໄດ້ເລຍ ໂດຍບໍ່ໄດ້ຍຸ້ງກັບ HDD ແຕ່ກັບການໃສ່ແບບນີ້ຄອມເກົ່າຈະບໍ່ມີບ່ອນໃສ່

ພາກສ່ວນແນະນຳ

ຈະເປັນຄຳແນະນຳ ແລະ ບົບຮຽນເພີ່ມເຕີມ

ຖ້າຕ້ອງການເລົ່ງໃຫ້ຈື່ເພື່ອໄປສອບເສັ່ງແນະນຳໃຫ້ເລົ່າແບບ
