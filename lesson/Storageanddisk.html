Storage and Disk

ການຮູ້ກຽວກັບ ພື້ນທີ່ຈັດເກັບ ແລະ ຮາກໄດ້ disk ຈະຊ່ວຍໃຫ້ເຮົາຈັດການກັບຄອມພີວເຕີຂອງເຮົາ ໃຫ້ມີພື້ນທີ່ວ່າງເພີ່ມຂື້ນ ແລະ ຈັດພື້ນທີ່ຕ່າງໆໃຫ້ເປັນລະບຽບຫຼາຍຂື້ນ

ຫົວໜ່ວຍຂອງຂໍ້ມູນ

ການຮູ້ຫົວໜ່ວຍຂອງຂໍ້ມູນຈະຊ່ວຍໃຫ້ຮູ້ວ່າຈະຕ້ອງຈັດສັນພື້ນທີ່ແບບໃດ

ຫົວໜ່ວຍຂໍ້ມູນຫຼັກໆທີ່ເຮົາໄດ້ໃຊ້ໆກັນມາຈະມີປະມານນີ້:

  • 1 byte = 8 bit

  • 1 KB (Kilobyte) = 1024 byte

  • 1 MB (Megabyte) = 1024 KB

  • 1 GB (Gigabyte) = 1024 MB

  • 1 TB (Terabyte) = 1024 GB

ຄວາມຮູ້ພື້ນຖານກຽວກັບ Disk

ຈະມາແນະນຳກຽວກັບການສ້າງ disk ໃຫມ່ , ການລົບ ແລະ ການເລື່ອກບ່ອນເກັບຂໍ້ມູນ ຈະຊ່ວຍໃຫ້ການຈັດລະບຽບຂໍ້ມູນຕ່າງໆງ່າຍ ແລະ ເປັນລະບຽບຫຼາຍຂື້ນ

Disk ຫຼຶ ຊ່ອງເກັບຂໍ້ມູນຕ່າງໆ

Disk ແມ່ນພື້ນທີ່ຈັດເກັບຂອງຄອມພິວເຕີ ທີ່ເຮົາສາມາດແບ່ງ ແລະ ຈັດການໃຫ້ມັນເປັນລະບຽບ ຫຼື ເໝາະກັບການໃຊ້ງານຂອງເຮົາໄດ້

ໂດຍ disk ເລີ່ມຕົ້ນ ທີ່ເປັນ disk ທີ່ຫຼັກໃຊ້ໃນການເກັບຂໍ້ມູນ ບະບົບປະຕິບັດການ ຫຼື ລະບົບສຳຄັນຕ່າງໆແມ່ນຢູ່ disk C ສະນັ້ນເຮົາບໍ່ຄວນໄປຍຸງກັບ disk ນີ້

ການເລືອກຊ່ອງສຳຫຼັບເກັບຂໍ້ມູນຕ່າງໆ

ການເລືອກຊ່ອງ ຫຼື Disk ໃນການເກັບຂໍ້ມູນນັ້ນມີຄວາມສຳຄັນຫຼາຍ ເພາະຊ່ວຍໃຫ້ການຈັດລະບຽບຂໍ້ມູນໃນຄອມພິວເຕີເປັນລະບຽບ ແລະ ຄົ້ນຫາຂໍ້ມູນໄດ້ງ່າຍຂຶ້ນຫຼາຍ ຖ້າເຮົາແບ່ງ Disk ໄວ້ຢ່າງຖືກຕ້ອງ ຕົວຢ່າງ: ເກມ ຫຼື ໄຟລວຽກ ກໍຈະຊ່ວຍໃຫ້ຄອມພິວເຕີເປັນລະບຽບ ແລະ ໃຊ້ງານສະດວກຂຶ້ນຫຼາຍ

  • ແຍກ Disk ສຳລັບ Windows ແລະ ໂປຣແກຣມສຳຄັນຕ່າງໆ ເຊັ້ນ: Microsoft Office

  • ເກມ ແລະ ໂປຣແກຮມເສີມຕ່າງໆ

  • ໄຟລວຽກ ຫຼື ໂປຣເຈກຕ່າງໆ

ການແບ່ງ ແລະ ການລົບແລ້ວດຶ້ງກັບ

  • ຂັ້ນຕອນໃນການແບ່ງຊ່ອງເກັບຂໍ້ມູນ ຫຼື ແບ່ງ disk ມີດັ່ງນີ້:

    ໄປທີ່ This pc → ເລືອກ Computer ໃນແຖບເມນູດ້ານທິງ → ເລືອກ manage ໃນແຖບເມນູຊ້າຍສຸດ → ເລືອກ Disk managemet ໃນເມນູດ້ານຂວາ → ແລ້ວເຮົາຈະເຫັນ disk ຫຼາຍໆອັນ ແນະນຳຖາຈະແບ່ງໃຫ້ແບ່ງຈາກອັນທີ່ຫຼາຍທີ່ສຸດ → ຄິກຂວາໃສ່ຊ່ອງ ຫຼຶ disk ທີ່ຕ້ອງການ → ກົດ shrik volume → ເລືອກຂະໜາດຕາມຕ້ອງການໃນແຖບ Enter the amount of space to shrink in MB (ການໃຊ່ນະໜາດນັ້ນນັບເປັນ MB ຖ້າຈະແບ່ງເປັນເທົ່າໃດ GB ກໍ່ຕ້ອງເອົາຂະຫຼາດ GB ທີ່ຕ້ອງການໄປ ຄູນ 1024 ເຊັ້ນ: 10 GB = 10 x 1024 = 10240 MB ) → ກົດ shrink → ແລ້ວເຮົາຈະໄດ້ ເພື້ອທີ່ວ່າງໆມາ ທີ່ຈະຂຽນວ່າ Unallocated → ຄິກຂວາທີ່ທີ່ຊ່ອງທີ່ເຮົາແຕກມາ ທີ່ເປັນ disk Unallocated → ເລືອກ New Simple Volume → ກົດ Next → ກົດ Next → ໃນໜ້າທີ່ເມນູ Assign Drive Letter or Path ເປັນການຕັ້ງຊື່ ຊ່ອງ ຫຼື disk ທີ່ເຮົາແຍກອອກມາ → ເລືອກຢູ່ໃນແຖບ Assign the following drive letter → ກົດ Next → ກົດ Next → ກົດ Finish → ແລ້ວເົຮາຈະໄດ້ disk ໃຫນ່ມາທີ່ເປັນຕົວອັກສອນຕາມທີ່ເຮົາຕັ້ງ

  • ຂັ້ນຕອນໃນການລົບຊ່ອງທີ່ເຮົາແບ່ງມາ ແລະ ດື່ງພື້ນທີ່ກັບຄືນ:

    ໄປທີ່ This pc → ເລືອກ Computer ໃນແຖບເມນູດ້ານທິງ → ເລືອກ manage ໃນແຖບເມນູຊ້າຍສຸດ → ເລືອກ Disk managemet ໃນເມນູດ້ານຂວາ → ຄິກຂວາ disk ທີ່ເຮົາຕ້ອງການລົບ (ຈະລົບໄດ້ແຕ່ສະເພາະ disk ທີເຮົາສ້າງຂື້ນມາເທົ່ານັ້ນ ຖ້າາເປັນ disk C ຫຼື disk ຫຼັກຈະບໍ່ສາມາດລົບໄດ້) → ເລືອກ Delete Volume ມັນຈະມີຂື້ນມາເຕືອນ ໃຫ້ກົດ Yes → ແລ້ວເຮົາຈະໄດ້ Unallocated ມາເປັນ Disk ວ່າງໆ → ແລ້ວໃຫ້ໄຳປຄິກຂວາທີ່ Disk ຕົ້ນທີ່ເຮົາໄປແບ່ງມາເຊັ້ນ: ແບ່ງມາຈາກ disk C ກໍໄປກົດຄິກຂວາທີ່ Disk C → ກົດ Extend Volume → ກົດ Next → ກົດ next → ກົດ Finish

ພາກສ່ວນແນະນຳ

ການຈັດການກັບ Disk ຈະເຮັດໃຫ້ຄອມພີວເຕີຂອງເຮົາເປັນລະບຽບຂື້ນຫຼາຍ ແລະ ການຈັດການກັບ disk ທີ່ເຮົາບໍ່ໄດ້ໃຊ້ກໍ່ຈະເຮັດໃຫ້ຄອມເຮັດວຽກໄດ້ດີຂື້ນ
