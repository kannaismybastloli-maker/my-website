ຄວາມຮູ້ພື້ນຖານກຽວກັບ File

ຄວາມຮູ້ດ້ານນີ້ຈະຊ່ວຍໃຫ້ເຂົ້າໃຈ ແລະ ຮູ້ຈັກ ກັບ file ປະເພດຕ່າງໆແລະການໃຊ້ງານຂອງມັນໄດ້

ນາມສະກຸນ ຫຼຶ ປະເພດຕ່າງໆຂອງ File

ໃນລະບົບຄອມພີວເຕີນັ້ນ ສາມາດເກັບຂໍ້ມູນໄດ້ຫຼາຍປະເພດເຊັ້ນ: ຕົວໜັງສື , ເຄື່ອງໝາຍ , ຮູບ , ວິດີໂອ , ສຽງ ແລະ ໄຟເອກະສານຕ່າງໆ

ການຈັດປະເພດ File ໂດຍທົ່ວໄປ

ໂດຍພື້ນຖານແລ້ວ ເຮົາຈະແບ່ງໄຟລ ອອກເປັນ 5 ປະເພດດັ່ງນີ້:

  • Media files : ໄຟລປະເພດມີເດຍຕ່າງໆເຊັ້ນ:

    • ຮູບພາບ : .jpg , .png , .gif ແລະ ອື່ນໆ

    • ສຽງ : .mp3 , .mid , .wnv ແລະ ອື່ນໆ

    • ວິດີໂອ : .mp4 , .flv , .mov ແລະ ອື່ນໆ

  • General Document Files : ໄຟລເອກະສານທົ່ວໆໄປ:

    • .txt

    • .pdf

  • Office Files : ໄຟລຕ່າງໆ ທີ່ຖືກສ້າງມາຈາກ microsoft office

    • Microsoft world : .dox

    • Microsoft Excel : .xlsx

    • Microsoft Powerpoint: .pptx

  • progaram / application file : ໄຟລທີ່ເປັນ ໄຟລ program ຫຼຶ application ທີ່ສາມາດ install ຫຼຶ ເປິດໃຊ້ງານໄດ້

    • .exe : .dox

  • Compressed Files : ໄຟທີ່ຖຶກປິດອັດໄວແລ້ວ ຄວນແຕກໄຟອອກມາກ່ອນເພື່ອໃຫ້ເບີ່ງ ຫຼື ໃຊ້ງາຍໄດ້ແບບສະດວກສະບາຍ ຈະເປັນໄຟລທີມີນານມສະກຸນແບບນີ້:

    • .zip

    • .rar

ການຄົ້ນຫາ ຫຼຶ ຊອກຫາໄຟລຕ່າງໆ

ຖ້າໃນຄອມພີວເຕີຂອງເຮົາຖ້າມີໄຟລຫຼາຍການຊອກຫາໄຟລຕ່າງໆກໍ່ຈະຢາກຂື້ນ ສະນັ້ນເຮົາຄວນຮູ້ວິທີຊອກຫາຂໍ້ມູນຕ່າງເພື່ອໃຫ້ງາຍໃນການຊ່ອງຫາຂໍ້ມູນທີ່ເຮົາຕ້ອງການ

ການຄົນຫາໄຟລຕ່າງໆນັ້ນສາມາດເຮັດໄດ້ໃນຫຼາຍແບບ ແຕ່ໃນນີ້ຈະມາແນະນຳໃນ 3 ແບບຄື:

    • ຄົ້ນຈາກຊື່ເຕັມເລຍເຊັ້ນ : lesson.dox

    • ຄົ້ນຫາຈາກນາມສະກຸນເຊັ້ນ : *.dox

    • ຄົ້ນຫາຈາກຕົວຂື້ນຄົນ ຫຼື ຊື່ບາງສ່ວນ : A*.*.dox

ພາກສ່ວນແນະນຳ

ຈະເປັນຄຳແນະນຳ ແລະ ບົບຮຽນເພີ່ມເຕີມ

ການຈັດໄຟລຕ່າງໆໃຫ້ເປັນລະບຽບ

ເຮົາຄວນທີ່ຈະຕ້ອງຈັດລະແບບຂອງໄຟລຕ່າງໆໃຫ້ເປັນລະບຽບ ທີ່ສຸດ ຢ່າງນ້ອຍໆຄວນຈັດຕາມປະເພດຕ່າງໆເພື່ອໃຫ້ງ່າຍຕໍ່ການຊອກຫາ ແລະ ນຳໃຊ້ໃນເວລາທີ່ຕ້ອງການ
